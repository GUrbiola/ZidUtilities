# CommonCode.Web

Web utilities and helpers for web-based applications.

## Description

This library provides common web-related utilities and extensions for ASP.NET and web applications. It serves as a foundation for web-specific functionality that complements the core CommonCode library.

## Features

- Web request/response handling utilities
- HTTP context helpers
- Web configuration management
- Session and state management utilities
- Web-specific extension methods

## Dependencies

- System.Web
- CommonCode (base library)

## Target Framework

.NET Framework 4.8

## Usage

This library is designed to be used in web applications that require common web functionality and utilities. Import this library in your ASP.NET projects for enhanced web development capabilities.
